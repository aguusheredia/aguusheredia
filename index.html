Linwo Video

Aplicación web. Maquetación con HTML5 y CSS3. JavaScript. Linwo Video es una aplicación web que muestra un catálogo de películas separadas por género. Dichas películas cuentan con una breve descripción de la misma. El catálogo es traido desde la api pública 'https://yts.mx/api/v2/'.
